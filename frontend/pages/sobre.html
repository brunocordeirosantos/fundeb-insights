Sobre o Projeto

Uma plataforma analítica para transformar dados públicos da educação brasileira em inteligência acessível — para gestores, pesquisadores e quem toma decisões.

Três perguntas que guiam a análise

O projeto foi construído em torno de questões concretas de política pública

01

Distribuição

Como os recursos do FUNDEB são distribuídos entre os 5.596 municípios, estados e regiões do país?

02

Impacto

Existe relação entre o investimento por aluno e os resultados educacionais medidos pelo IDEB?

03

Eficiência

Quais municípios entregam desempenho acima do esperado dado o nível de investimento recebido?

Pipeline de dados

Da fonte pública à análise interativa

Extração

Download automatizado dos arquivos XLSX do FNDE e CSV do IBGE via scripts Python. IDEB obtido manualmente do portal INEP.

extract_fundeb.py extract_ibge.py
→

Limpeza

Normalização de colunas, tratamento de nulos, padronização de códigos IBGE (7 dígitos), remoção de duplicatas e validação de tipos.

clean_ideb.py
→

Integração

Join das três bases pelo código IBGE do município. Cálculo de receita per capita. Geração do arquivo analítico com 5.596 linhas e 97% de cobertura.

merge_datasets.py
→

Exposição

API REST em FastAPI carrega a base em memória e serve 8 endpoints de consulta, ranking, correlação e agregação por UF para o frontend.

Limitações conhecidas

  • IDEB disponível apenas para municípios com número mínimo de alunos avaliados — ~3% sem nota
  • Dados populacionais de 2021 (estimativa) — não refletem crescimento recente
  • Análise de correlação é observacional — não implica causalidade
  • FUNDEB 2026 são receitas previstas, não executadas
